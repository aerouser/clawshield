## How to Contribute

### Reporting Bugs
- Use GitHub Issues
- Include: OS, Node version, skill scanned, output

### Suggesting Features
- Open an issue first to discuss
- Security features are always welcome

### Pull Requests
1. Fork the repo
2. Create branch: `git checkout -b feature/your-feature`
3. Commit: `git commit -m 'Add amazing feature'`
4. Push: `git push origin feature/your-feature`
5. Open PR

### Code Style
- Use existing formatting
- Add comments for complex logic
- Update README if needed

### Security Rules
Want to add a new detection pattern?
1. Add rule to `src/scanner/engine.js`
2. Add test fixture in `tests/fixtures/`
3. Update rule count in README

## Questions?
Open an issue or join our Discord.
